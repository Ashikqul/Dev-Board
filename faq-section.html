What is event delegation in the DOM?
Event delegation হল একটি টেকনিক যেখানে একটি parent element-এ ইভেন্ট লিসেনার যোগ করা হয়, এবং ইভেন্ট bubbling এর মাধ্যমে child element গুলোর ইভেন্ট হ্যান্ডেল করা হয়
What is event bubbling in the DOM?
Event bubbling হচ্ছে একটি DOM ইভেন্টের প্রক্রিয়া যেখানে ইভেন্টটি একটি child element থেকে parent element এর দিকে প্রপাগেট হয়।
How do you create, add, and remove elements using JavaScript?
JavaScript-এ উপাদান তৈরি, যোগ এবং মুছে ফেলার জন্য কিছু পদ্ধতি রয়েছে। নতুন উপাদান তৈরি করতে document.createElement() ব্যবহার করা হয়। একটি নতুন উপাদান যোগ করতে appendChild() বা insertBefore() ব্যবহার করা হয়, যা একটি উপাদানকে প্যারেন্ট উপাদানের মধ্যে যোগ করে। আর একটি উপাদান মুছে ফেলতে removeChild() বা remove() ব্যবহার করা হয়, যা নির্দিষ্ট উপাদানকে তার প্যারেন্ট থেকে সরিয়ে ফেলে।
